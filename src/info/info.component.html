How does it work?

The spreadsheets are compared cell by cell. The result of the comparisson contains the following information:

  • Cells that only exist in the file on the left.

  • Cells that only exist in the file on the right.

  • Cells that only exist in both files, but have different values.


Is it safe?

The most important feature of this tool is that it does not store or hold record of any data. There is no database linked to the website and the server does not log any messages. Not even for error or exception handling.


About and privacy

The tool does not store any data and does not use cookies. All loging is deactivated. The files' contents are sent to the server responsible for comparison over https (encrypted). Before the server responds with the differences, the files' contents are automatically deleted. The files' contents are also deleted if there was an error during comparison.
